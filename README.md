# Eventos Estrella Event Management Website

A responsive static website project for event management(dummy company). This project showcases skills in web development and graphic design, featuring custom illustrations converted from Adobe Illustrator to web-ready assets.

## Overview

The website presents a fictional event management company called Eventos Estrella, offering comprehensive services including event planning, lighting, sound, decorations, photography, videography, entertainment, and catering. It includes interactive sections with a gallery, services slideshow, contact form with validation, and responsive navigation.

## Key Features

- **Responsive Design**: Adapts seamlessly across desktop, tablet, and mobile devices
- **Interactive Slideshow**: JavaScript-powered gallery and services showcase with navigation controls
- **Form Validation**: Client-side validation for contact form using vanilla JavaScript
- **Custom Typography**: Integrated custom OTF font files for branding (Blue Highway family)
- **Image Gallery**: Lazy-loaded event images with grid layout
- **Mobile Menu**: Collapsible navigation for mobile devices using jQuery
- **Dark Theme**: Modern dark color scheme with theme variables

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with flexbox layout, responsive media queries, and theme color variables
- **JavaScript**: Interactive functionality including slideshow, form validation, and mobile menu toggle
- **jQuery 1.8.3**: DOM manipulation for mobile menu interactions
- **Adobe Illustrator**: Original design templates for logos, icons, and graphics

## How to View

1. Clone or download the repository
2. Navigate to the `Events/web/` directory
3. Open `index.html` in your preferred web browser
4. The website is fully static and requires no server setup
